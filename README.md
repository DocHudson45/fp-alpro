# Web Project Discovery & Analysis Tool

A Next.js application designed to help agencies and freelancers analyze client requests, generate project scopes, and produce design concepts efficiently.

## Features

- **Project Discovery**: Simplify data collection from clients through intuitive flows.
- **AI Analysis**: Identify technical complexities and core business logic quickly.
- **Moodboard Generation**: Instantly mock up visual directions based on project specs.
- **Prompt Architect**: Export tailored prompts for development environments.

## Local Setup

1. Setup your environment variables (refer to `.env.example` if available).
2. Run `npm install`
3. Run `npm run dev` to start the local development server.

## Deployment

This app is optimized for Vercel deployment. Make sure all environment variables are correctly configured in your Vercel project settings before deploying.
